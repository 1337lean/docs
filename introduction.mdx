# Introduction

These docs cover two related products: the browser-based buffer.lol diagnostics toolbox and IP Lens, a native iPhone networking app that is coming soon.

buffer.lol provides quick network, IP, web, and developer checks. Use it to inspect domains, endpoints, certificates, ports, public IP details, browser signals, and everyday encoded data without opening a full monitoring suite.

## What buffer.lol checks

<CardGroup cols={2}>
  <Card title="Network diagnostics" icon="network">
    Inspect DNS records and resolver agreement, email DNS health, HTTP and security headers, TLS certificates, redirects, uptime, ports, robots.txt, and sitemap hints.
  </Card>
  <Card title="IP intelligence" icon="map-pin">
    Detect your public IP and inspect country, RDAP network data, and ASN ownership.
  </Card>
  <Card title="Browser utilities" icon="code">
    Format JSON, parse URLs, decode JWTs, generate hashes and UUIDs, convert timestamps, and test regex patterns locally.
  </Card>
  <Card title="Connection diagnostics" icon="activity">
    Measure browser-to-buffer.lol HTTPS latency and stability, or trace server-side network hops through the restricted diagnostics worker.
  </Card>
</CardGroup>

## IP Lens for iPhone

IP Lens brings public IP lookup, DNS-over-HTTPS, authoritative RDAP registration, offline IPv4 and IPv6 subnet calculation, bounded batch lookup, saved results, and local CSV/JSON export into a native iPhone app.

IP Lens does not use the buffer.lol diagnostics API or operate its own lookup backend. Online requests go directly from the iPhone to the selected IP data provider, DNS resolver, or authoritative registry. Start with the [IP Lens overview](/ip-lens/overview).

## Find a tool

Choose **Find a tool** or press <kbd>Command</kbd>+<kbd>K</kbd> on macOS or <kbd>Ctrl</kbd>+<kbd>K</kbd> on Windows and Linux. Search by a tool name, command, or keyword, then use the arrow keys and <kbd>Enter</kbd> to open it. Quick access updates with up to five recently opened tools in that browser.

Diagnostic result pages also suggest related tools. When it is safe and useful, the next tool opens with the same public target already filled in.
