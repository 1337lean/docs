# buffer.lol docs

Mintlify documentation for `docs.buffer.lol`.

## Local preview

```bash
npm install
npm run dev
```

Open `http://localhost:3333`.

## Mintlify setup

1. Push this folder to the GitHub repository you want Mintlify to use.
2. In Mintlify, connect that GitHub repository.
3. Use the repository root as the docs path because `docs.json` is in this folder.
4. Add `docs.buffer.lol` in Mintlify's custom domain settings.
5. Add the DNS record Mintlify gives you at your DNS provider.

The canonical URL is already set to `https://docs.buffer.lol` in `docs.json`.
