# IP Lens overview

<img src="/images/ip-lens/app-icon.png" alt="IP Lens app icon" width="112" />

IP Lens is a focused networking toolkit built natively for iPhone. It combines public IPv4 and IPv6 lookup with DNS inspection, authoritative registration data, offline subnet calculations, bounded batch analysis, saved results, and local exports.

IP Lens is **coming soon** for iPhone models running iOS 17 or later. It is planned as a $4.99 one-time App Store purchase with no subscription, advertisements, or account requirement.

## What you can do

| Tool | Purpose |
| --- | --- |
| IP Lookup | Inspect approximate location, ASN, ISP, organization, timezone, and network details for a public IPv4 or IPv6 address |
| DNS Lookup | Query `A`, `AAAA`, `CNAME`, `MX`, `NS`, `TXT`, and `PTR` records through a selected DNS-over-HTTPS resolver |
| Registration Lookup | Review domain, IP, and ASN registration data from the authoritative RDAP registry |
| Subnet Calculator | Calculate IPv4 and IPv6 networks, ranges, masks, prefixes, and capacity without a network request |
| Batch Lookup | Analyze up to 50 unique public IP addresses in one run |
| Saved results | Reopen lookup history, favorites, and explicitly saved DNS, RDAP, and batch captures offline |
| Export | Prepare CSV or JSON files locally and choose where to share them with the iOS share sheet |

## How requests work

IP Lens does not send lookups through buffer.lol and does not operate a developer-controlled lookup backend. Online requests go directly from the iPhone to the service needed for the selected tool:

- Free IP lookups use `ipwho.is`; personal-key mode uses `ipwhois.pro`.
- DNS queries go to exactly one selected resolver: Cloudflare or Quad9.
- Registration queries go to the authoritative RDAP registry selected with IANA bootstrap data.
- Subnet calculations and export generation stay on the device.

This architecture is separate from the same-origin diagnostics API documented in the [buffer.lol API reference](/api-reference/overview).
